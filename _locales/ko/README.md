# NotebookLM 툴킷

NotebookLM 웹사이트의 사용 경험을 향상시키고, 바로가기 키, 강화된 버튼, NotebookLM 웹사이트 빠른 열기 버튼을 제공합니다.

## 기능

- 마인드맵
  - 「복사」 버튼을 제공하여 클릭하면 펼쳐진 모든 노드의 텍스트를 Markdown 형식으로 복사할 수 있습니다.

- 노트
  - NotebookLM 웹사이트의 노트 페이지에서 노트의 전체 내용을 빠르게 복사할 수 있는 버튼을 추가합니다.

- 사용자 정의 음성 요약 프롬프트 저장
  - 사용자 정의 음성 요약에서 입력한 프롬프트 내용을 자동 저장
  - 확장 프로그램 아이콘을 우클릭하여 "음성 요약의 사용자 정의 프롬프트 기록"을 선택하여 저장된 프롬프트 확인
  - 프롬프트 관리 지원: 보기, 펼치기/접기, 삭제 기능
  - 중복 저장 방지, 최근 50개 프롬프트 자동 관리
