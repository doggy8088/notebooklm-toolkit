# NotebookLM 万能工具箱

提升 NotebookLM 网站的使用体验，提供快捷键、增强按钮与快速打开 NotebookLM 网站的按钮。

## 功能

- 思维导图
  - 在思维导图右下角会增加一个按钮，点击后可以展开所有节点。
  - 按下 `Ctrl+Alt+A` 快捷键，可以一次展开所有思维导图节点。

- 笔记
  - 在 NotebookLM 网站的笔记页面，新增一个按钮，可以快速复制笔记完整内容。

- 自定义语音摘要提示保存
  - 自动保存您在自定义语音摘要中输入的提示内容
  - 右键点击扩展图标选择「语音摘要的自定义提示记录」查看已保存的提示
  - 支持提示管理：查看、展开/收起、删除功能
  - 防止重复保存，自动管理最近 50 个提示
