# NotebookLM Werkzeugkasten

Verbessern Sie Ihr NotebookLM-Website-Erlebnis mit Tastenkombinationen, erweiterten Schaltflächen und schnellem Zugang zur NotebookLM-Website.

## Funktionen

- Mind Map
  - Bietet eine „Kopieren"-Schaltfläche, mit der Sie alle erweiterten Knotentexte im Markdown-Format kopieren können.

- Notizen
  - Fügt eine Schaltfläche auf der NotebookLM-Notizenseite hinzu, um den vollständigen Notizinhalt schnell zu kopieren.

- Benutzerdefinierte Audio-Zusammenfassung Prompt-Speicherung
  - Speichert automatisch Prompt-Inhalte, die Sie in benutzerdefinierten Audio-Zusammenfassungen eingeben
  - Rechtsklicken Sie auf das Erweiterungssymbol und wählen Sie „Benutzerdefinierte Audio-Zusammenfassung Prompt-Verlauf", um gespeicherte Prompts anzuzeigen
  - Unterstützt Prompt-Verwaltung: Anzeigen, Erweitern/Einklappen und Löschfunktionen
  - Verhindert doppelte Speicherungen, verwaltet automatisch die neuesten 50 Prompts
