# NotebookLM ツールキット

NotebookLM ウェブサイトの使用体験を向上させ、キーボードショートカット、拡張ボタン、NotebookLM ウェブサイトへの素早いアクセスボタンを提供します。

## 機能

- マインドマップ
  - マインドマップの右下角にボタンを追加します。クリックするとすべてのノードを展開できます。
  - `Ctrl+Alt+A` ショートカットキーを押すと、すべてのマインドマップノードを一度に展開できます。

- ノート
  - NotebookLM ウェブサイトのノートページに、ノートの完全な内容を素早くコピーできるボタンを追加します。

- カスタム音声要約プロンプト保存
  - カスタム音声要約で入力したプロンプト内容を自動保存
  - 拡張機能アイコンを右クリックして「音声要約のカスタムプロンプト履歴」を選択し、保存されたプロンプトを表示
  - プロンプト管理をサポート：表示、展開/折りたたみ、削除機能
  - 重複保存を防止し、最新の50個のプロンプトを自動管理
