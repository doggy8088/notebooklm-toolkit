# Toolkit NotebookLM

Migliora la tua esperienza sul sito NotebookLM con scorciatoie da tastiera, pulsanti migliorati e accesso rapido al sito NotebookLM.

## Funzionalità

- Mappa mentale
  - Fornisce un pulsante "Copia" che consente di copiare tutto il testo dei nodi espansi in formato Markdown.

- Note
  - Aggiunge un pulsante nella pagina delle note di NotebookLM per copiare rapidamente il contenuto completo della nota.

- Salvataggio prompt riassunto audio personalizzato
  - Salva automaticamente il contenuto dei prompt che inserisci nei riassunti audio personalizzati
  - Clicca con il tasto destro sull'icona dell'estensione e seleziona "Cronologia prompt personalizzati riassunto audio" per visualizzare i prompt salvati
  - Supporta la gestione dei prompt: visualizzazione, espansione/compressione e funzioni di eliminazione
  - Previene salvataggi duplicati, gestisce automaticamente gli ultimi 50 prompt
