# NotebookLM 萬能工具箱

提升 NotebookLM 網站的使用體驗，提供快速鍵、增強按鈕與快速開啟 NotebookLM 網站的按鈕。

## 功能

- **心智圖**
  - 在心智圖右下角會增加一個按鈕，點擊後可以展開所有節點。
  - 按下 `Ctrl+Alt+A` 快速鍵，可以一次展開所有心智圖節點。

- **自訂語音摘要提示保存** 🆕
  - 自動保存您在自訂語音摘要中輸入的提示內容
  - 右鍵點擊擴充功能圖示選擇「查詢自訂提示」查看已保存的提示
  - 支援提示管理：檢視、展開/收起、刪除功能
  - 防止重複保存，自動管理最近 50 個提示
  - 詳細說明請參考 [自訂提示功能說明](CUSTOM_PROMPTS_FEATURE.md)
